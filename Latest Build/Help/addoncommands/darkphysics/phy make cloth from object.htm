PHY MAKE CLOTH FROM OBJECT

Create a cloth from the specified object.

  Syntax 
phy make cloth from object ID, object
  Parameters 

ID
identification number of the cloth 

object
identification number of the object

  Returns

This command does not return a value.

  Description

In certain situations it may be more appropriate to have a non rectangular shape to represent the cloth. In cases such as this it is possible to provide a custom object that will be used for the cloth. For example, it may be that a model of a cape has been provided, this can be loaded into the scene and then phy make cloth from object called for it and then the cloth built. After this point the cape object will behave and respond in the form of a cloth object.
