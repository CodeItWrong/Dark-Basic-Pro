PHY GET TRIGGER TYPE

Find out the type of trigger.

  Syntax 
integer = phy get trigger type ( ID  )
  Parameters 

ID
identification number of the trigger

  Returns

a value of 0 indicates a box trigger while a value of 1 means a sphere trigger

  Description

Use this command to determine what kind of trigger the object is. Currently only two types of triggers are supported being boxes and spheres.
