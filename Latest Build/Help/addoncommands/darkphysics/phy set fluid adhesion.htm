PHY SET FLUID ADHESION

Sets the adhesion property of a fluid object.

  Syntax 
phy set fluid adhesion ID, value#
  Parameters 

ID
identification number of the fluid object 

value#
new adhesion level

  Returns

This command does not return a value.

  Description

This is the friction of the fluid regarding the surface of a static shape. It must be between 0 and 1. A value of 1 will cause the particle to lose its velocity tangential to the surface normal of the shape at the collision location i.e. it will not slide along the surface. A value of 0 will preserve the particles velocity in the tangential surface direction i.e. it will slide without resistance on the surface.
