PHY FREE CLOTH VERTEX

Free a cloth vertex from the point it was attached to.

  Syntax 
phy free cloth vertex ID, vertex
  Parameters 

ID
identification number of the cloth object 

vertex
the vertex index that is to be freed

  Returns

This command does not return a value.

  Description

A vertex of a cloth object can be reattached to an object or point in world space at a later stage using phy attach cloth vertex to shape and phy attach cloth vertex.
