PHY SET FLUID REST DENSITY

Set the fluid rest density property.

  Syntax 
phy set fluid rest density ID, value#
  Parameters 

ID
identification number of the fluid object 

value#
new rest density property

  Returns

This command does not return a value.

  Description

Controls the target density for the fluid. As an example water is about 1000. Even if the particle system is simulated without particle interactions, this property defines indirectly in combination with rest particles per metre the mass of one particle ( mass = rest density / rest particles per meter ^ 3 ) ). The particle mass has an impact on the repulsion effect on emitters and rigid bodies.
