PHY SET DYNAMIC FRICTION

Set the scaling of dynamic friction for all objects.

  Syntax 
phy set dynamic friction value#
  Parameters 

value#
new dynamic friction scale

  Returns

This command does not return a value.

  Description

This command lets the user scale the magnitude of the dynamic friction applied to all objects. By default this is a value of 1.0 and the range is from 0 to infinity.
