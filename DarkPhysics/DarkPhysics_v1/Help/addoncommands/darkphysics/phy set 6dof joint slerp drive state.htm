PHY SET 6DOF JOINT SLERP DRIVE STATE

Control if slep drive is used on the joint. The slerp drive state is used to drive the joint along the shortest spherical arc

  Syntax 
phy set 6dof joint slerp drive state ID, state
  Parameters 

ID
identification number of the joint 

state
a value of 1 will switch slerp drive on while a value of 0 will switch it off.

  Returns

This command does not return a value.
