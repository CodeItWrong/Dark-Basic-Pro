PHY SET FLUID EMITTER RANDOM POSITION

Sets the range of positions from where the emitter can emit.

  Syntax 
phy set fluid emitter random position ID, x#, y#, z#
  Parameters 

ID
identification number of the fluid emitter 

x#
variation on X axis 

x#
variation on Y axis 

x#
variation on Z axis

  Returns

This command does not return a value.

  Description

Once set the position of emitted particles will be randomized based on the Z value. The values have to be positive and describe the maximal random particle displacement in each dimension.
