PHY SET VEHICLE SUSPENSION DAMPING

Set the suspension damping property for the vehicle.

  Syntax 
phy set vehicle suspension damping ID, value#
  Parameters 

ID
identification number of the vehicle 

value#
new suspension damping property

  Returns

This command does not return a value.

  Description

The damping property of a vehicle is used as a way of slowing movement on a spring over time. The higher this value is then the faster the spring will come to a stop whereas a lower value will allow it much more freedom. The default value for the damper is 0.0 and the range extends from 0.0 to infinity.
