PHY SET EMITTER END COLOR

Control the color that particles fade to when they reach the end of their life span.

  Syntax 
phy set emitter end color ID, red, green, blue
  Parameters 

ID
identification number of the emitter object 

red
value between 0 - 255 indicating quantity of red 

green
value between 0 - 255 indicating quantity of green 

blue
value between 0 - 255 indicating quantity of blue

  Returns

This command does not return a value.

  Description

The setting of the end color can be useful to have the effect of particles fading out as they reach the end of their lifespan. For example, you may set the end color to black and as the particles reach the end of their life they will fade out and disappear which may be preferable to them simply vanishing.
