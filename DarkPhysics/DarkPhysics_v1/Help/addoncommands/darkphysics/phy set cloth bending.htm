PHY SET CLOTH BENDING

Control whether bending is in use for a cloth object.

  Syntax 
phy set cloth bending ID, state
  Parameters 

ID
identification number of the cloth object 

state
a value of 1 switches bending on while a value of 0 turns bending off

  Returns

This command does not return a value.

  Description

If a cloth can be bent then its bending stiffness property can be controlled to handle the level of bendiness for a cloth. This command determines if bending is enabled or disabled.
