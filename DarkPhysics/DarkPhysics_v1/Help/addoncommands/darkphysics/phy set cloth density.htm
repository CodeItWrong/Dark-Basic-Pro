PHY SET CLOTH DENSITY

Set the density property of a cloth object.

  Syntax 
phy set cloth density ID, value#
  Parameters 

ID
identification number of the cloth object 

value#
the new density

  Returns

This command does not return a value.

  Description

The density of a cloth object is the mass per area. The default value for this is 1.0 and the range goes from 0.0 to infinity. Use this command to set the density of a specified cloth object.
