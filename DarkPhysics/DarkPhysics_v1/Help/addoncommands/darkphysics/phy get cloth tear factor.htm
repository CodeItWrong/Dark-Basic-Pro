PHY GET CLOTH TEAR FACTOR

Determine the tear factor property of the cloth object.

  Syntax 
float = phy get cloth tear factor ( ID )
  Parameters 

ID
identification number of the cloth object

  Returns

tear factor property

  Description

The tear property of a cloth defines the elongation factor that causes the cloth to tear. By default this value is set to 1.5 and can be in the range from 0.0 to infinity. Use this command to find out the tear factor property of the specified cloth object.
