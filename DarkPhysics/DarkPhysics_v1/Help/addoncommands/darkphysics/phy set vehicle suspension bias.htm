PHY SET VEHICLE SUSPENSION BIAS

Set the suspension bias on the vehicle.

  Syntax 
phy set vehicle suspension bias ID, value#
  Parameters 

ID
identification number of the vehicle 

value#
new bias level

  Returns

This command does not return a value.

  Description

The vehicle has springs attached to the wheels and this property defines the target value ( angle / position ) of the spring where the spring force is zero. Its angular property is in the range of -3.14 to 3.14 and the positional property is in the range of -infinity to infinity. The default value is -0.25.
