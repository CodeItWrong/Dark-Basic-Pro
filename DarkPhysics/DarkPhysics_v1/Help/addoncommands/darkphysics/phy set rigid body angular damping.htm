PHY SET RIGID BODY ANGULAR DAMPING

Set the angular damping of a rigid body.

  Syntax 
phy set rigid body angular damping ID, value#
  Parameters 

ID
ID number of the rigid body 

value#
new angular damping property for the rigid body, a value of 0.0 indicates no damping, the parameter must be positive.

  Returns

This command does not return a value.

  Description

The angular damping property can be used as a way of bringing objects to a stop over time by limiting their angular momentum.
