PHY BUILD MATERIAL

Build a material and make it ready to be used.

  Syntax 
phy build material ID
  Parameters 

ID
the ID number of the material that you want to build, if a material does not exist with this ID number the command will fail and an error message will be displayed

  Returns

This command does not return a value.

  Description

After a material has been created and all of its properties set it is then ready to be built. Building the material will complete its construction and enter it into the physics simulation. After this stage the material can be applied to rigid bodies.
