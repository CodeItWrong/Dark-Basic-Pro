PHY SET FLUID EMITTER DIMENSIONS

Control the size of the emitter on the X and Y axis.

  Syntax 
phy set fluid emitter dimensions ID, x#, y#
  Parameters 

ID
identification number of the fluid emitter 

x#
size of the emitter on the X axis 

y#
size of the emitter on the Y axis

  Returns

This command does not return a value.

  Description

Changing the dimensions of the emitter on the X and Y axis can be used to alter how particles are emitted.
