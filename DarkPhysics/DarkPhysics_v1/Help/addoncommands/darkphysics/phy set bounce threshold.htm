PHY SET BOUNCE THRESHOLD

Control the bounce threshold of objects.

  Syntax 
phy set bounce threshold value#
  Parameters 

value#
the new bounce threshold property

  Returns

This command does not return a value.

  Description

This command is used to control the bounce thresholds at which objects will bounce. A contact with a relative velocity below this threshold will not bounce. The default setting for this threshold is -2.0. The range for this property is -infinity to 0.
