PHY GET RIGID BODY LOCAL POINT VELOCITY X

Returns the local point velocity of the rigid body on the X axis.

  Syntax 
float = phy get rigid body local point velocity x ( ID, x#, y#, z# )
  Parameters 

ID
ID number of the rigid body 

x#
local point on X axis 

y#
local point on Y axis 

z#
local point on Z axis

  Returns

Local point velocity on X axis.
