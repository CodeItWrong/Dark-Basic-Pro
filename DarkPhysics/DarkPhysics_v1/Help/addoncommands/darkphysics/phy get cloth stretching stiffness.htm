PHY GET CLOTH STRETCHING STIFFNESS

Determine the stretching stiffness property of the cloth object.

  Syntax 
float = phy get cloth stretching stiffness ( ID )
  Parameters 

ID
identification number of the cloth object

  Returns

stretching stiffness property

  Description

The stretching stiffness property of a cloth controls how easy it is for the cloth to be stretched. By default this is set to a value of 1.0 and the range is from 0.0 to 1.0. Use this command to find out the current stretching stiffness of the specified cloth object.
