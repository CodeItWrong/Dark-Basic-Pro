PHY SET SPHERE JOINT SWING LIMIT

Defines the angle which the joint can rotate away from the swing axis.

  Syntax 
phy set sphere joint swing limit ID, value#, resitution#, hardness#
  Parameters 

ID
identification number of the joint 

value#
the angle / position beyond which the limit is active 

restitution#
limit bounce 

hardness#
limit can be made softer by setting this to less than 1 ( not yet implemented )

  Returns

This command does not return a value.
