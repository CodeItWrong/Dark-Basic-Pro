PHY SET FLUID EMITTER VELOCITY

Set the velocity of particles when they are being emitted.

  Syntax 
phy set fluid emitter velocity ID, velocity#
  Parameters 

ID
identification number of the fluid emitter 

velocity#
velocity property of particles being emitted

  Returns

This command does not return a value.

  Description

When setting the velocity a low value will result in particles being emitted and not travelling very far. A high value used here will give the particles a lot of movement as they get emitted.
