PHY SET FLUID EMITTER MAX PARTICLES

Sets the maximum number of particles that a fluid emitter has.

  Syntax 
phy set fluid emitter max particles ID, max
  Parameters 

ID
identification number of the fluid emitter 

max
maximum number of particles to be used by the fluid emitter

  Returns

This command does not return a value.

  Description

Increasing the maximum number of particles to a very high number will in turn increase the level of complexity for the simulation. It is recommended to try and find a good balance between the number of particles and the frame rate. In most cases a value of around 2000 may be suitable or even far fewer.
