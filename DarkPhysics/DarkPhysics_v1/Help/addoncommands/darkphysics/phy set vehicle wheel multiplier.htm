PHY SET VEHICLE WHEEL MULTIPLIER

Controls how fast the wheels spin when moving.

  Syntax 
phy set vehicle wheel multiplier ID, multiplier#
  Parameters 

ID
identification number of the vehicle 

multiplier#
default value is 15.0, increase or decrease to get the right scale based on your vehicle

  Returns

This command does not return a value.

  Description

As the vehicle moves along certain calculations are moved to spin the wheels. Dependant on the size of the vehicle it may be that the default value does not fit in line with expectations. To resolve this simple set the vehicle wheel multiplier and increase or decrease the value until the right results are found. The default value for this property is 15.0 and its range is from -infinity to infinity.
