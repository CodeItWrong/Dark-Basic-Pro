PHY SET FLUID MOTION LIMIT

Set the motion limit property of a fluid.

  Syntax 
phy set fluid motion limit ID, value#
  Parameters 

ID
identification number of the fluid object 

value#
the new motion limit property

  Returns

This command does not return a value.

  Description

This property is the maximal distance a particle is allowed to travel within one timestep of the simulation. It is relative to the rest spacing of the particles, which is defined by the command phy set fluid rest per metre. The maximal travel distance = motion limit multiplier / rest particles per metre.
