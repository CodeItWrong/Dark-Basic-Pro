PHY SET PULLEY JOINT RATIO

Set the transmission ratio e.

  Syntax 
phy set pulley joint ratio ID, ratio#
  Parameters 

ID
identification number of the joint 

ratio#
ratio level#

  Returns

This command does not return a value.

  Description

g. if set to 2 would cause the 2nd object attached to move twice as far as the 1st object attached to the joint.
