PHY SET FLUID REST PER METRE

Set the fluid rest per metre property.

  Syntax 
phy set fluid rest per metre ID, value#
  Parameters 

ID
identification number of the fluid object 

value#
new rest per metre property

  Returns

This command does not return a value.

  Description

This property is the particle resolution given as particles per linear meter measured when the fluid is in its rest state ( relaxed ). Even if the particle system is simulated without particle interactions, this property defines the emission density of the emitters.
