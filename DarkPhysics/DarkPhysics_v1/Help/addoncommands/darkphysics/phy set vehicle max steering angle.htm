PHY SET VEHICLE MAX STEERING ANGLE

Set the maximum steering angle for the wheels.

  Syntax 
phy set vehicle max steering angle ID, value#
  Parameters 

ID
identification number of the vehicle 

value#
maximum steering angle

  Returns

This command does not return a value.

  Description

When the wheels of a vehicle turn they are constrained by the maximum steering angle. Use this command to control the limits of their turning. Note that the value# parameter is to be specified in radians.
