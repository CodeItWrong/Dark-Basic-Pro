PHY BUILD CLOTH

Use this command to build a cloth object.

  Syntax 
phy build cloth ID
  Parameters 

ID
identification number of the cloth object to be built

  Returns

This command does not return a value.

  Description

In order to use a cloth object a necessary step is to build it and that is carried out by making a call to the command phy build cloth. It is only when the cloth has been built that it is created and entered into the simulation for processing. Making a cloth and not calling this command will result in the cloth object not appearing on screen and in the simulation.

Note that after the cloth has been built certain properties cannot be changed such as its size.
