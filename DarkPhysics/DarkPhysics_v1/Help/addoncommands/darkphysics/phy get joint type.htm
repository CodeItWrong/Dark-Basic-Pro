PHY GET JOINT TYPE

Determine the type of joint.

  Syntax 
return intger = phy get joint type ( ID )
  Parameters 

ID
the identification number of the joint

  Returns

The following values can be returned: 0, sphere joint 1, revolute joint 2, prismatic joint 3, cylindrical joint 4, fixed joint 5, distance joint 6, point in plane joint 7, point on line joint 8, pulley joint
