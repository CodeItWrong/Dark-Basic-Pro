PHY SET EMITTER START COLOR

Control the color of particles as they are emitted.

  Syntax 
phy set emitter start color ID, red, green, blue
  Parameters 

ID
the identification number of the emitter object 

red
value between 0 - 255 indicating quantity of red 

green
value between 0 - 255 indicating quantity of green 

blue
value between 0 - 255 indicating quantity of blue

  Returns

This command does not return a value.
