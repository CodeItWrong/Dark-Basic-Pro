PHY SET VEHICLE WHEEL POWER

Control which wheels are powered.

  Syntax 
phy set vehicle wheel power ID, front, rear
  Parameters 

ID
identification number of the vehicle 

front
set to 1 to enable power in the front wheels while 0 disables power 

rear
set to 1 to enable power in the rear wheels while 0 disables power

  Returns

This command does not return a value.

  Description

When a vehicle is simulated it may be appropriate to have a the front or rear wheels powered or all of them. Use this command to specify which vehicles are powered. For example, to power the front and rear wheels then simply call the command passing in the ID of the vehicle and then use a value of 1 for the front and rear parameters.
