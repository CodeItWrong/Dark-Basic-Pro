PHY MAKE PULLEY JOINT

Make a new pulley joint.

  Syntax 
phy make pulley joint ID, A, B 
phy make pulley joint ID, A, B, x1#, y1#, z1#, x2#, y2#, z2#, x3#, y3#, z3#, x4#, y4#, z4#, distance#, stiffness#, ratio#
  Parameters 

ID
identification number of the joint, joint ID numbers are unique 

A

  Returns

This command does not return a value.

  Description

There are two approaches to creating a pulley joint. The command can be called passing in a full set of parameters to create the joint in one line ( some defaults are used when this is the case ). If you choose this option then the joint is built straight away. Alternatively make a pulley joint passing in three parameters and then proceed to set properties such as the local anchor etc and finally build the joint using phy build pulley joint. The second option for creating a pulley joint may be preferable when you want complete control over the settings.

Note that joint ID numbers are unique and not shared with any other ID ranges therefore it is perfectly valid to have a model loaded into ID 1 and a joint with an ID of 1.
