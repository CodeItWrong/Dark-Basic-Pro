PHY SET MAX ANGULAR VELOCITY

Set the maximum angular velocity for rigid bodies.

  Syntax 
phy set max angular velocity value#
  Parameters 

value#
new maximum angular velocity property

  Returns

This command does not return a value.

  Description

Use this command to set the maximum angular velocity permitted for all rigid bodies. This is useful for solving errors within the simulation that can be introduced as a result of rigid bodies rotating very quickly which may lead to undesirable results. The default value for this property is 7 and the range is from 0 to infinity.
