PHY SET VEHICLE STEERING VALUE

Set the steering value for the vehicle wheels.

  Syntax 
phy set vehicle steering value ID, value#
  Parameters 

ID
identification number of the vehicle 

value#
new steering value

  Returns

This command does not return a value.

  Description

The steering value is the final property applied to wheels to rotate them. It takes its value from the steering angle along with some other internal values.
