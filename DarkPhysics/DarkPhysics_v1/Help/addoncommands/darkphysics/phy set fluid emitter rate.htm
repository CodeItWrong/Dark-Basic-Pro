PHY SET FLUID EMITTER RATE

Set the rate at which particles are emitted.

  Syntax 
phy set fluid emitter rate ID, rate#
  Parameters 

ID
identification number of the fluid emitter 

rate#
number of particles being emitted per second

  Returns

This command does not return a value.

  Description

The rate specifies how many particles are emitted per second but bear in mind the rate is only considered if the emitter is using a constant flow rate ( as controlled by the emitter type property ).
