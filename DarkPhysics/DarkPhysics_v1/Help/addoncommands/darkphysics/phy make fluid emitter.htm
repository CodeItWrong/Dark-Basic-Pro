PHY MAKE FLUID EMITTER

Create a new fluid emitter object.

  Syntax 
phy make fluid emitter ID
  Parameters 

ID
identification number of the fluid object, these numbers are shared with those of Basic 3D

  Returns

This command does not return a value.

  Description

A fluid emitter is used as a way of emitting fluid from a position and can be used to simulate such things as fluid coming out of a pipe. Once a fluid emitter has been made it is necessary to set its properties using commands such as phy set fluid emitter type. After properties have been set the fluid can be built using the command phy build fluid.
