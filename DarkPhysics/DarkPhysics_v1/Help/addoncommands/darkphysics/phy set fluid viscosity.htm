PHY SET FLUID VISCOSITY

Set the fluid viscosity.

  Syntax 
phy set fluid viscosity ID, value#
  Parameters 

ID
the identification number of the fluid object 

value#
new viscosity property

  Returns

This command does not return a value.

  Description

The viscosity of the fluid defines its viscous behaviour. Higher values will result in a honey-like behaviour. Viscosity is an effect which depends on the relative velocity of neighboring particles; it reduces the magnitude of the relative velocity. This property must be positive.
