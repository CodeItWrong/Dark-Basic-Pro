PHY SET 6DOF JOINT LINEAR LIMIT

Set the linear limit on a joint.

  Syntax 
phy set 6dof joint linear limit ID, value#, restitution#, spring#, damping#
  Parameters 

ID
identification number of the joint 

value#
the angle / position beyond which the limit is active 

restitution#
controls the amount of bounce when the joint hits a limit 

spring#
if greater than 0, the limit is soft, i.e. a spring pulls the joint back to the limit 

damping#
if spring# is greater than 0 then this is the damping of the spring

  Returns

This command does not return a value.
