PHY SET EMITTER RATE

Controls the emitter rate.

  Syntax 
phy set emitter rate ID, rate#
  Parameters 

ID
identification number of the emitter object 

update#
the new rate

  Returns

This command does not return a value.

  Description

In conjunction with the phy set emitter update property this command controls how quickly new particles are spawned. When a new particle is spawned ( as controlled by phy set emitter update ) the new timer is set to this rate. By increasing this timer you can make it so that new particles take longer or faster to spawn.
