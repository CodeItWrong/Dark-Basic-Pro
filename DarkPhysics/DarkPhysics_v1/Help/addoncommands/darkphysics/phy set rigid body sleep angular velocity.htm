PHY SET RIGID BODY SLEEP ANGULAR VELOCITY

Control the angular velocity below which a rigid body may go to sleep.

  Syntax 
phy set rigid body sleep angular velocity ID, value#
  Parameters 

ID
ID number of the rigid body 

value#
new sleep angular velocity, range is from 0 to infinity

  Returns

This command does not return a value.

  Description

Sets the angular velocity below which a rigid body may go to sleep. Rigid bodies whose angular velocity is above this threshold will not be put to sleep. If the threshold value is negative then the velocity threshold is set using the default sleep angular velocity property.
