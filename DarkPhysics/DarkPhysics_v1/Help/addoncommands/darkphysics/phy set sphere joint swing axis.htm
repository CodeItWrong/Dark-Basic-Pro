PHY SET SPHERE JOINT SWING AXIS

Set the local axis.

  Syntax 
phy set sphere joint swing axis ID, x#, y#, z#
  Parameters 

ID
identification number of the joint 

x#
direction on the X axis 

y#
direction on the Y axis 

z#
direction on the Z axis

  Returns

This command does not return a value.

  Description

This command will set the swing axis for the specified rigid body attached to the sphere joint.
