PHY GET CLOTH BENDING STIFFNESS

Determine the bending stiffness property of a cloth object.

  Syntax 
float = phy get cloth bending stiffness ( ID )
  Parameters 

ID
identification number of the cloth object

  Returns

a value indicating the bending stiffness property

  Description

This property controls how easy it is for the cloth to bend. The range is between 0.0 and 1.0 with the default value being 1.0. Note that it is only used when bending is allowed for the cloth.
