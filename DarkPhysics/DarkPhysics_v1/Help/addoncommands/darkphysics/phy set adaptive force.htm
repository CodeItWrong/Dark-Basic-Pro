PHY SET ADAPTIVE FORCE

Enable or disable adaptive forces.

  Syntax 
phy set adaptive force value#
  Parameters 

value#
true or false value indicating whether adaptive forces are to be used

  Returns

This command does not return a value.

  Description

Used to enable adaptive forces to accelerate convergence of the solver. By default this property is set to true. To enable adaptive forces use a value of 1 and to disable a value of 0.
