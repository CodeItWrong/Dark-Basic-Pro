PHY SET EMITTER RANGE

Use this command to control the range over which particles are emitted.

  Syntax 
phy set emitter range ID, min x#, min y#, min z#, max x#, max y#, max z#
  Parameters 

ID
identification number of the emitter object 

min x#
minimum position to emit on X axis 

min y#
minimum position to emit on Y axis 

min z#
minimum position to emit on Z axis 

max x#
maximum position to emit on X axis 

max y#
maximum position to emit on Y axis 

max z#
maximum position to emit on Z axis

  Returns

This command does not return a value.

  Description

This command is used when you require particles to be emitted over a certain area rather than from one point. As an example this property can be controlled to create a rain or snow effect.
