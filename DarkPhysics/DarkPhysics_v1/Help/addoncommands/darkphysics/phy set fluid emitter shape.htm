PHY SET FLUID EMITTER SHAPE

Set the shape of the specified fluid emitter.

  Syntax 
phy set fluid emitter shape ID, shape
  Parameters 

ID
identification number of the fluid emitter 

shape
a value of 1 will use a rectangular shape while a value of 2 will set the emitter to be an ellipse

  Returns

This command does not return a value.

  Description

Currently two emitter shapes are supported: rectangular and elliptical. By default when an emitter is created it will be set to use a rectangular shape.
