PHY SET FIXED TIMING

Control the elapsed time between calls to updating the simulation.

  Syntax 
phy set fixed timing elapsedTime#
  Parameters 

elapsedTime#
time elapsed between each call to updating the simulation

  Returns

This command does not return a value.

  Description

When fixed timing is in use the default time between each update of the simulation is 1.0 / 60.0. In some cases this may not be appropriate therefore this command allows you to control this property.
