PHY SET VEHICLE MAX MOTOR

Set the maximum motor power.

  Syntax 
phy set vehicle max motor ID, value#
  Parameters 

ID
identification number of the vehicle 

value#
maximum motor power

  Returns

This command does not return a value.

  Description

When automatic driving mode is used it is necessary to set the maximum amount of force that the motor can have applied to it. When the up key is pressed the motor force will be set to the maximum motor power. By default this value is set to 360.0. This property has no effect when manual driving is used.
