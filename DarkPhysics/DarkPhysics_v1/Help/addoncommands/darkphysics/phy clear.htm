PHY CLEAR

Clears will physics objects from the scene.

  Syntax 
phy clear
  Parameters 

This command does not need any parameters.
  Returns

This command does not return a value.

  Description

The phy clear command is a useful way of wiping all rigid bodies, cloth objects, particles etc. from a scene in one simple call. Upon calling this command all entities in the physics engine will be removed and no rigid bodies etc. will exist. It may be faster to use this command when you want to free up resources rather than attempting to delete each one manually.
