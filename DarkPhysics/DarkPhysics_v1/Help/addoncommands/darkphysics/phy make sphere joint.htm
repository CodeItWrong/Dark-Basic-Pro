PHY MAKE SPHERE JOINT

Make a new sphere joint.

  Syntax 
phy make sphere joint ID, A, B 
phy make sphere joint ID, A, B, x#, y#, z#
  Parameters 

ID
identification number of the joint, joint ID numbers are unique 

A
the first rigid body object to be attached to the joint 

B
the second rigid body object to be attached to the joint 

x#
position on X axis 

y#
position on Y axis 

z#
position on Z axis

  Returns

This command does not return a value.

  Description

There are two approaches to creating a sphere joint. The command can be called passing in a full set of parameters to create the joint in one line ( some defaults are used when this is the case ). If you choose this option then the joint is built straight away. Alternatively make a sphere joint passing in three parameters and then proceed to set properties such as the local anchor etc and finally build the joint using phy build sphere joint. The second option for creating a sphere joint may be preferable when you want complete control over the settings.

Note that joint ID numbers are unique and not shared with any other ID ranges therefore it is perfectly valid to have a model loaded into ID 1 and a joint with an ID of 1.
