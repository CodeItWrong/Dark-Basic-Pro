PHY SET VEHICLE MASS

Set the mass of the vehicle.

  Syntax 
phy set vehicle mass ID, value#
  Parameters 

ID
identification number of the vehicle 

value#
new mass

  Returns

This command does not return a value.

  Description

By default the mass of the vehicle is set to a value of 80. This value can by overriden by calling this command. The range can be from 0.0 to infinity.
