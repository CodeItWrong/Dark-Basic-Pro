PHY SET FLUID EMITTER LIFETIME

Controls the lifetime of particles being emitted.

  Syntax 
phy set fluid emitter rate ID, lifetime#
  Parameters 

ID
identification number of the fluid emitter 

lifetime#
how long the particles will live for in seconds

  Returns

This command does not return a value.

  Description

This property is the time in seconds which an emitted particle lives for. If this is set to 0 then the particle will live until it comes into contact with a drain.
