PHY SET 6DOF JOINT DRIVE POSITION

Set the position to drive the joint towards.

  Syntax 
phy set 6dof joint drive position ID, x#, y#, z#
  Parameters 

ID
identification number of the joint 

x#
position on the X axis 

y#
position on the Y axis 

x#
position on the Z axis

  Returns

This command does not return a value.
