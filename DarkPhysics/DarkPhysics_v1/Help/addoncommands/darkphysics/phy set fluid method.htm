PHY SET FLUID METHOD

Use this command to control how the fluid is simulated.

  Syntax 
phy set fluid method ID, value#
  Parameters 

ID
the identification number of the fluid object 

value#
use a value of 0 to set simulation of inter particle forces, 1 does not simulate inter particle forces and 2 alternates between mode 0 and simple particles

  Returns

This command does not return a value.

  Description

This command defines whether or not particle interactions are considered in the simulation.
